# Banks

## 概要

連携サービス

## エンドポイント一覧

### GET /api/1/banks

**操作**: 連携サービス一覧の取得

**説明**: 概要 連携しているサービス一覧を取得する 定義 type bank_account : 銀行口座 credit_card : クレジットカード wallet : その他の決済口座

### GET /api/1/banks/{id}

**操作**: 連携サービスの取得

**説明**: 概要 連携しているサービスを取得する 定義 type bank_account : 銀行口座 credit_card : クレジットカード wallet : その他の決済口座
